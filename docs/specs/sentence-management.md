# Sentence Management — Feature Spec

## Overview

This spec defines how `tome-ms-language` manages **sentences** for the Language Learning module.

A sentence is a full phrase in the target language (e.g. Danish) along with its English translation. Sentences may come from two sources:

1. **Source extraction** — sentences found directly in a knowledge source (e.g. a Google Doc). These are extracted by `tome-ms-sources` during the knowledge extraction flow.
2. **Agent generation** — sentences synthesised on demand by an LLM agent in `tome-ms-sources`, based on words already in the vocabulary. These are identified by the special source type `"tome-agent"`.

Sentences are persisted in a dedicated MongoDB collection named **`sentences`**, separate from the `vocabulary` collection.

---

## Data Model

Each sentence entry represents a single phrase and its translation.

| Field             | Type             | Description                                                                                                                                                                   |
|-------------------|------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `id`              | string           | MongoDB ObjectId, generated on insert                                                                                                                                         |
| `language`        | string           | Target language code (e.g. `"danish"`)                                                                                                                                        |
| `sentence`        | string           | The sentence in the target language (e.g. Danish)                                                                                                                             |
| `translation`     | string           | The English translation of the sentence                                                                                                                                       |
| `createdAt`       | string           | ISO 8601 timestamp of when the entry was created                                                                                                                              |
| `knowledgeSource` | string           | The ID of the data source this sentence was extracted from, or `"tome-agent"` if the sentence was generated by an LLM agent. Used for provenance tracking.                    |

---

## API Endpoints

All paths are relative to the service base path `/tomelang`.

| Operation       | Method | Path                                           | Delegate        |
|-----------------|--------|------------------------------------------------|-----------------|
| PostSentence    | `POST` | `/sentences/{language}`                        | `PostSentence`  |
| PostSentences   | `POST` | `/sentences/{language}/batch`                  | `PostSentences` |
| GetSentences    | `GET`  | `/sentences/{language}`                        | `GetSentences`  |

## Endpoint Specifications

### POST `/sentences/{language}` — PostSentence

Adds a single sentence to the sentences collection.

#### Path Parameters

| Parameter  | Description                             |
|------------|-----------------------------------------|
| `language` | Target language (e.g. `"danish"`)       |

#### Request Body

```json
{
  "sentence": "Jeg kan godt lide at læse bøger.",
  "translation": "I like reading books.",
  "knowledgeSource": "664abc123def456789abcdef"
}
```

| Field             | Required | Description                                                            |
|-------------------|----------|------------------------------------------------------------------------|
| `sentence`        | Yes      | The sentence in the target language. Non-empty string.                 |
| `translation`     | Yes      | The English translation. Non-empty string.                             |
| `knowledgeSource` | Yes      | Data source ID, or `"tome-agent"` for AI-generated sentences.         |

#### Response — `201 Created`

```json
{
  "id": "664abc123def456789abcdef"
}
```

#### Error Cases

| Condition                                            | Status | Description                            |
|------------------------------------------------------|--------|----------------------------------------|
| Missing `sentence`, `translation`, or `knowledgeSource` | `400` | Required fields not provided           |
| Unknown / unsupported `language`                     | `400`  | Language is not in the supported list  |

---

### POST `/sentences/{language}/batch` — PostSentences

Inserts multiple sentences in one request. Follows the same best-effort batch strategy as `PostWords`.

#### Path Parameters

| Parameter  | Description                             |
|------------|-----------------------------------------|
| `language` | Target language (e.g. `"danish"`)       |

#### Request Body

```json
{
  "sentences": [
    {
      "sentence": "Jeg kan godt lide at læse bøger.",
      "translation": "I like reading books.",
      "knowledgeSource": "664abc123def456789abcdef"
    },
    {
      "sentence": "Hun bor i København.",
      "translation": "She lives in Copenhagen.",
      "knowledgeSource": "tome-agent"
    }
  ]
}
```

| Field       | Required | Description                              |
|-------------|----------|------------------------------------------|
| `sentences` | Yes      | Array of sentence objects (min length: 1) |

Each sentence object follows the same rules as `PostSentence`.

#### Batch Behaviour

- **Best-effort**: valid sentences are inserted; invalid ones are rejected individually.
- The operation does **not** abort on a partial failure.

#### Response — `207 Multi-Status`

```json
{
  "results": [
    {
      "sentence": "Jeg kan godt lide at læse bøger.",
      "status": "created",
      "id": "664abc123def456789abcdef"
    },
    {
      "sentence": "Hun bor i København.",
      "status": "error",
      "reason": "missing_field"
    }
  ]
}
```

| `status` value | Meaning                                         |
|----------------|-------------------------------------------------|
| `"created"`    | Sentence was successfully inserted              |
| `"error"`      | Sentence was rejected; `reason` field explains why |

Possible `reason` values: `"missing_field"`.

#### Error Cases

| Condition                        | Status | Description                            |
|----------------------------------|--------|----------------------------------------|
| `sentences` array is missing/empty | `400` | Request is malformed                   |
| Unknown / unsupported `language` | `400`  | Language is not in the supported list  |

### GET `/sentences/{language}` — GetSentences

#### Path Parameters

| Parameter  | Description                             |
|------------|-----------------------------------------|

#### Error Cases

## Random Word Sampling Endpoint

The sentence generation flow in `tome-ms-sources` requires a set of vocabulary words to use as "seed" material for LLM generation. This endpoint provides that capability.

### GET `/vocabulary/{language}/words/sample` — SampleWords

Returns a random sample of vocabulary words for the specified language.

#### Path Parameters

| Parameter  | Description                             |
|------------|-----------------------------------------|
| `language` | Target language (e.g. `"danish"`)       |

#### Query Parameters

| Parameter | Required | Default | Description                                              |
|-----------|----------|---------|----------------------------------------------------------|
| `n`       | No       | `20`    | Number of words to sample. If fewer words exist in the vocabulary than `n`, all words are returned. |

#### Response — `200 OK`

```json
{
  "language": "danish",
  "words": [
    {
      "id": "664abc123def456789abcdef",
      "english": "dog",
      "translation": "hund",
      "createdAt": "2026-04-01T10:00:00.000Z",
      "knowledgeSource": "src-dataset-42"
    }
  ]
}
```

The sample is **random** — there is no ordering guarantee. Each call may return a different subset of words.

#### Error Cases

| Condition                        | Status | Description                            |
|----------------------------------|--------|----------------------------------------|
| Unknown / unsupported `language` | `400`  | Language is not in the supported list  |
| `n` is not a positive integer    | `400`  | Invalid `n` parameter                  |

---

## Business Rules

1. **Supported languages:** Same as vocabulary — defined in service configuration. Currently only `"danish"`.
2. **Sentence uniqueness:** There is uniqueness constraint on `(language, sentence)`. The same sentence may NOT be inserted more than once. A Deduplication check must happen. It can happen before or after insertion, based on a performance evaluation to be done before implementing. 
3. **`knowledgeSource` is always required:** Even for AI-generated sentences, the value must be `"tome-agent"`. Empty strings are rejected.
4. **`createdAt`:** Set server-side at insert time. Not updatable.
5. **`userId` is not stored** on sentences (unlike `sources`). Sentences are shared knowledge for the language; they are not user-scoped.

---

## Out of Scope

- Editing or deleting sentences.
- Pagination of `GetSentences` results.
- Filtering sentences by `knowledgeSource`.
- Soft deletes.
- Sentence practice / flashcard sessions (may be specified separately).
- Audio pronunciation or TTS for sentences.
